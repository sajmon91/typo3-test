=============================================================
Speaking URL fragments (anchors)
=============================================================

.. only:: html

   :Classification:
      content_slug

   :Version:
      |release|

   :Language:
      en

   :Description:
      Adds a slug field for human-readable anchors ("domain.com/page/#my-section") to TYPO3 content elements. By default, this anchor is rendered as the header's id attribute.
